O mne

Som freelance webdesignérka.

Pomáham malým firmám v SK a CZ s tým, aby ich web pracoval rovnako tvrdo ako oni samé.

Príbeh

Začalo to ako záľuba.

Prvý web som postavila pri škole pre kamarátku, ktorá mala kaderníčku v Topoľčanoch. Stránku z roku 2013 nahradila moderná verzia, ktorá jej priniesla dvojnásobok rezervácií za pol roka. Vtedy som pochopila, že web pre malú firmu nie je luxus — je to nástroj, ktorý buď pracuje, alebo nepracuje.

Dnes pomáham desiatkam malých živnostníkov a firiem od kaderníctiev cez fyzio ambulancie po fotografov. Každý web robím sama, od prvého rozhovoru až po publikáciu domény. Žiadny outsourcing, žiadne agentúrne reči.

Hodnoty

Čo robím a čo nerobím

Fixné ceny

Termíny dodržujem

✗

Bez závislostí

Nezviazané na drahé CMS, ktoré platíte mesačne. Web vlastníte vy, hosting vlastníte vy.

✗

Bez upsell-u

Nesnažím sa vám pridať blog, e-shop a 5 jazykov, ak to nepotrebujete. Robíme len to, čo má zmysel.

✗

Bez agentúrnych rečí

Žiadne „synergia", „pivot", „leverage". Volá sa to webovka a má pomôcť vašim klientom nájsť vás.

Stack

S čím pracujem

Pre transparentnosť — toto sú nástroje, ktoré používam každý deň.

Vývoj

  • HTML5 / CSS3 / JavaScript — vanilla, žiadny zbytočný framework
  • Astro / Next.js — pre väčšie weby s blogom alebo e-shopom
  • Tailwind CSS — pre rýchlu konzistentnosť
  • Cursor IDE / VS Code — pre kódovanie

Dizajn & nasadenie

  • Figma — pre wireframy a hero sekcie
  • Netlify / Vercel — bezplatný hosting pre väčšinu projektov
  • GitHub Pages — pre tých, čo už majú GitHub účet
  • Cloudflare — DNS, doména, ochrana

Stretnime sa.
